# 📊 Dividend Stock Analysis System（存股分析系統）

本專案為一套以 Python 開發的長期投資分析工具，  
透過整合公開市場資料與爬蟲技術，自動蒐集個股歷史財務數據與股利政策，  
並提供可操作的桌面介面，協助使用者快速篩選適合的投資標的。

---

## 🚀 專案動機

在進行存股投資時，需要頻繁查詢：

- 個股歷年股利發放
- 每股盈餘（EPS）
- 殖利率
- 股價變化

傳統方式需手動查詢多個網站，不僅耗時，也容易因判斷標準不一致而影響決策。

因此，本專案將「選股流程系統化」，  
把投資條件轉換為可重複執行的規則，提升分析效率與一致性。

---

## ⚙️ 核心功能

### 🔹 自動化資料蒐集
- 擷取公開市場資料（股價、殖利率等）
- 爬取個股歷史股利與財務數據

### 🔹 資料處理與儲存
- 使用 SQLite 建立本地資料庫
- 設計資料更新機制（避免重複抓取，提升效能）

### 🔹 選股條件篩選
- 支援依殖利率、價格、股號、股名等條件搜尋
- 提供類似 SQL 的查詢邏輯進行篩選

### 🔹 桌面應用介面（Tkinter）
- 顯示股票列表與篩選結果
- 可查看個股詳細資訊（歷年股利 / 季報 EPS）
- 即時更新股價與殖利率

---

## 🧠 分析邏輯

本系統以長期投資（存股）為核心，透過以下條件進行篩選：

- 穩定股利發放紀錄
- 合理或偏高的殖利率
- 長期 EPS 表現
- 股價與報酬率關係

並將上述條件轉換為規則模型，  
降低人工選股成本，並提升決策一致性。

---

## 🏗️ 技術架構

- Python
- requests / BeautifulSoup（資料爬取）
- SQLite（資料儲存）
- Tkinter（桌面 GUI）
- threading（背景資料更新）

👉 採用「資料層 / 邏輯層 / 介面層」分離設計
